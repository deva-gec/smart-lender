## Export to PDF

Choose one method:

### Pandoc (recommended)

```bash
pandoc docs/Smart_Lender_Project_Documentation.md \
  -o docs/Smart_Lender_Project_Documentation.pdf \
  --pdf-engine=xelatex
```

### VS Code

1. Install the **Markdown PDF** extension
2. Open `Smart_Lender_Project_Documentation.md`
3. Right-click → **Markdown PDF: Export (pdf)**

### Typora

File → Export → PDF

### Online

Upload the `.md` file to https://www.markdowntopdf.com/

---

After exporting, commit the PDF alongside the Markdown file for submission.
